# BB-8 阶段 6：可拆球壳与线束维护

> 本阶段冻结维护空间和接口合同，不代表球壳模具、密封圈型材或锁扣供应商已经冻结。正式壳体加工仍需材料试片和样件装配。

## 赤道球壳接口

| 项目 | 当前包络 | 状态 |
|---|---:|---|
| 球体外径 | 508 mm | 银幕参考复刻基准 |
| 赤道密封中心直径 | 494 mm | 预留约7 mm径向壳体/法兰空间 |
| 密封圈截面包络 | Ø5 mm | 型材待壳体材料确定 |
| 锁扣 | 8个，45°等分 | 供应商未冻结 |
| 单锁扣目标夹紧力 | 80 N | 设计合同，需实测 |
| 外观分缝 | 赤道连续分缝 | 已在Blender外观模型中表达 |

锁扣必须使用球内可释放、带防松或保险结构的方案。任何一个锁扣脱开时，不允许其零件落入驱动轮与球壳之间。

## 线束合同

- 每台IG42E设置一条动力线和一条编码器线；模型中共12段显式线束和4个维护连接器。
- 动力连接器合同：Molex 3.96 mm、2针；编码器合同：JST-PH 2.0 mm、4针，以Cytron当前商品页为依据。
- 动力线最低弯曲半径30 mm，编码器线最低弯曲半径20 mm。
- 线束经过底盘边缘处必须增加耐磨套；不得依赖胶带作为唯一固定方式。
- Blender审计用48 mm驱动轮保守包络检查每个线束网格顶点，要求不小于10 mm轮子间隙。

## 开壳顺序

1. 物理急停，断开主电池和保险丝；确认轮子不能通电。
2. 移除磁性头部并放入非磁工具托盘，避免夹伤。
3. 将球体放入带软垫的防滚支架，使赤道分缝水平。
4. 对角、分两轮释放8个锁扣，禁止从单侧连续拆完。
5. 上半壳抬起不超过50 mm，先观察是否有线束跨越分缝。
6. 断开Molex动力和JST编码器维护连接器，再完全移走上半壳。
7. 维护完成后反向装配，检查密封圈无扭曲、无夹线，再对角闭合锁扣。

## 实物验证门槛

- 壳体开合20次后，分缝错台、密封圈压缩永久变形和锁扣松动必须记录。
- 向球壳内放置无源假线束，手动转轮10圈不得出现擦碰或卷线。
- 低压限流供电下逐台电机架空运行，观察连接器、线束和锁扣振动。
- 完成淋水/粉尘测试前，不声明任何IP等级。
- 壳体材料、厚度和锁扣实测后，必须重新计算球壳质量、转动惯量和质心。
